Puli Extensions
===============

.. toctree::
   :hidden:

   extensions/twig

Puli_ features various extensions that integrate the library with other tools
and libraries. If you haven't got a working Puli repository yet, read
:doc:`getting-started`.

If you don't know what Puli is or why you should use it, read :doc:`at-a-glance`
first.

The following table points you to the documentation of the individual extensions:

=========  ===================================================================================  =====================================  ============
Tool       Description                                                                          Documentation                          Version
=========  ===================================================================================  =====================================  ============
Composer_  The `Puli plugin for Composer`_ builds repositories from composer.json definitions.  :doc:`repository-management/composer`  1.0.0-alpha1
Twig_      The `Twig extension`_ lets you access templates via Puli paths.                      :doc:`extensions/twig`                 1.0.0-dev
Symfony_   The `Puli bridge`_ and `Puli bundle`_ integrate Puli with Symfony.                   tbd                                    1.0.0-dev
Assetic_   You can create Assetic assets with Puli paths using the `Assetic extension`_.        tbd                                    1.0.0-dev
Pash_      The Pash shell lets you interactively browse Puli repositories.                      tbd                                    1.0.0-dev
=========  ===================================================================================  =====================================  ============
